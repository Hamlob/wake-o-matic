Detailed Description

Class that takes history of eyeStatus values and decides whether the person is awake, sleeping or not detected.

Currently, its only looking at the most prevalent value in the buffer holding the history.

USAGE:

Firstly, load the eye status values one by one using

See also
load(), then call
detect(), which returns status of the driver.
